Sobre la aplicación

Bueno, es hora de ponerse serios.
Esta es una aplicación web creada por Odei Alba, con la intención de simplificar el proceso de crear facturas.
Con esta aplicación se podrán añadir las facturas a una base de datos para después acceder a ellas, editarlas, eliminarlas, o incluso crear archivos pdf y excel con estilos predefinidos.

Esta aplicación consta con el primer commit en GitHub el 29 de septiembre del 2015. Es la segunda versión de la aplicación y hay dos repositorios diferentes con la primera y la segunda versión, ya que para crear la segunda versión era necesario editar la mayoría de los ficheros.

GitHub Versión Inicial: https://github.com/oalba/factura
GitHub Versión Final: https://github.com/oalba/facturanav

*Nota: Todos los datos que puedan influir en la privacidad de los usuarios están ocultos con asteriscos.
